# ABAR Dashboard

아바르 업체를 위한 통합 매출 및 매입 분석 대시보드입니다.

## 주요 기능
- **업체 자동 수집**: EOMS 사이트에서 크롤링된 219개의 업체 리스트 초기화
- **보안 로그인**: 접속 시 비밀번호 확인 (기본값: `abar1234`)
- **분석 탭**: 월별 매출, 매입, 광고비, 수수료, 기타비용 분석
- **쇼핑몰/홈쇼핑 구분**: 각 채널별 ROAS 및 수익 비중 분석
- **보고서 출력**: PDF 형태로 월별 분석 보고서 저장 및 출력
- **데이터 관리**: 거래처 및 월별 데이터의 추가/수정/삭제/저장

## 기술 스택
- **Vite + React**
- **Vanilla CSS** (Glassmorphism 디자인)
- **Recharts** (데이터 시각화)
- **jsPDF & html2canvas** (보고서 출력)

## 사용 방법
1. 프로젝트 폴더에서 `npm run dev` 실행
2. 브라우저에서 접속 후 비밀번호 `abar1234` 입력
3. 상단 '거래처 관리'에서 업체 리스트 확인
4. '데이터 입력' 탭에서 월별 실적 입력
5. '심층 분석' 탭에서 보고서 확인 및 PDF 출력
